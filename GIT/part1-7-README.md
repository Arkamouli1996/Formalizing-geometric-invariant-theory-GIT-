# `part1-7.lean` — overview

Plain-language summary of what each section proves. For precise hypotheses and statements, see the Lean file.

---

## Part 1 — graded invariants (`GradedCheck_S1`)

**THRM:** Given a graded `k`-algebra `R` with grading `𝒜`, a group `G` acting on `R` by `k`-algebra automorphisms `ρ`, and the hypothesis that the action **preserves** every graded piece, then the fixed subalgebra `R^G` is **internally graded**: it is the direct sum over degrees `d` of “homogeneous of degree `d` ∧ fixed,” so each invariant decomposes uniquely into invariant homogeneous parts.

**Also proved along the way:** the graded projection commutes with `ρ`; each graded component of an invariant is invariant; the forget map from the fixed degree-`d` submodule into `𝒜 d` is injective.

---

## Part 2 — irrelevant ideal and finite type (`IrrelevantIdeal_FiniteType_S2`)

**LEMMA:** Given a commutative ℕ-graded `k`-algebra `R` whose **irrelevant ideal** (the ideal generated by all positive-degree graded pieces) is **finitely generated** as an ideal of `R`, then `R` is **of finite type as an `𝒜 0`-algebra** (finitely generated as an algebra over the degree-0 piece).

**THRM:** Given the same finite-generation hypothesis on the irrelevant ideal, and in addition that **`𝒜 0` is finitely generated as a `k`-algebra**, then **`R` is finitely generated as a `k`-algebra**.

**Note:** Without finite generation of `𝒜 0` over `k`, finite generation of the irrelevant ideal does **not** by itself imply finite generation of `R` over `k`.

## Part 3 — extended positive invariants in a Noetherian ring (`RGplus_finitely_generated_S3`)

**THRM:** Given a **Noetherian** commutative ring `R` and an ideal `extendedRGplus` of `R` (intended as an extended invariant ideal), then that ideal is **finitely generated**, and there exists a **finite** subset of `R` whose span equals `extendedRGplus`.

---

## Part 4 — generators inside a prescribed set (`RGplus_generators_S4`)

**THRM:** Given a subset `RGplusSet ⊆ R` whose ideal span equals `extendedRGplus`, and given that **`extendedRGplus` is finitely generated** as an ideal, then there exists a **finite** set `s ⊆ RGplusSet` such that **the ideal generated by `s` is still `extendedRGplus`**.

---

## Part 5 — Reynolds / GIT spanning (`RGplus_generators_S5`)

**PROPERTY: `ReynoldsGITSpanProperty toR ρ s`** means: for every `f : A`, if `toR f` lies in the `R`-ideal generated by `s`, then `f` lies in the `A`-ideal generated by `{ ρ(x) | x ∈ s }`. Not proven, necessary for later thrm.

**LEMMA:** Given that **`Ideal.span s = extendedRGplus`**, that **`RGplusA = comap toR extendedRGplus`**, that **`ReynoldsGITSpanProperty toR ρ s`**, and that **`f ∈ RGplusA`** for some **`f : A`**, then **`f` lies in the `A`-ideal generated by `{ ρ(x) | x ∈ s }`**.

**THRM (`RGplusA_fg_of_reynolds`):** Given that `s` spans `extendedRGplus`, **`RGplusA = comap toR extendedRGplus`**, **`ρ(x) ∈ RGplusA` for all `x ∈ s`**, and **`ReynoldsGITSpanProperty toR ρ s`**, then **`RGplusA` is a finitely generated ideal of `A`**, generated by `{ ρ(x) | x ∈ s }`.

## Partt 6 - the Reynolds rewrite 

**Theorem (`reynolds_rewrite`):** If `f ∈ A` satisfies `toR(f) = ∑ xᵢ · cᵢ` in `R`
for some `xᵢ ∈ s` and coefficients `cᵢ ∈ R`, and if each `xᵢ` has a lift `x̃ᵢ ∈ A`
with `toR(x̃ᵢ) = xᵢ`, then applying `ρ` to both sides and using the Reynolds identity
gives `f = ∑ x̃ᵢ · ρ(cᵢ)` in `A`. That is, `f` is an `A`-linear combination of the
lifts `x̃ᵢ` with coefficients `ρ(cᵢ) ∈ A`.

**Lemma (`mem_ideal_span_lift_of_reynolds`):** Under the same hypotheses, `f` lies in
the `A`-ideal generated by `{ x̃ | x ∈ s }`.

**Theorem (`RGplusA_le_span_lift_of_reynolds`):** If every element of an ideal
`RGplusA ⊆ A` can be written as `toR(f) = ∑ xᵢ · cᵢ` with `xᵢ ∈ s`, then the entire
ideal `RGplusA` is contained in the `A`-ideal generated by the lifts of `s`, giving a
concrete finite generating set for `RGplusA`.

## Part 7 — The Fixed Subalgebra is Finitely Generated (`FixedSubalgebra_FiniteType_S7`)

**Theorem (`fixedSubalgebra_finiteType`):** Let `k` be a field and `A` a commutative
`k`-algebra carrying an ℕ-grading `𝒜G`. Suppose:
- the **degree-0 piece** `𝒜G(0)` is **finitely generated as a `k`-algebra**, and
- the **irrelevant ideal** of `A` (generated by all positive-degree homogeneous
  elements) is **finitely generated as an ideal of `A`**.

Then **`A` is finitely generated as a `k`-algebra**.
